# AI-Powered Smart Invisibility Cloak Using Computer Vision

## Overview

This project implements an AI-powered Smart Invisibility Cloak using Computer Vision, Deep Learning, Gesture Recognition, and Voice Control.

Unlike traditional invisible cloak projects that rely on color detection, this system uses YOLOv8 Human Segmentation to accurately identify and remove a person from the camera feed in real time.

## Features

* YOLOv8 Human Segmentation
* AI-Based Invisibility Effect
* Background Replacement
* MediaPipe Hand Gesture Recognition
* Voice Commands using Speech Recognition
* Real-Time Processing
* Multithreaded Voice Control

## Gesture Commands

* Open Palm → Visible
* Closed Fist → Invisible

## Voice Commands

* "visible" → Visible
* "invisible" → Invisible
